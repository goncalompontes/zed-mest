# zed-mest

[Zed](https://zed.dev) editor support for the [Mest](https://github.com/goncalompontes/mestc) programming language.

Provides syntax highlighting via Tree-sitter.

## Installing

### As a dev extension

1. Open Zed
2. Open the extensions panel (<kbd>Ctrl+Shift+X</kbd>)
3. Click **Install Dev Extension**
4. Select this directory
